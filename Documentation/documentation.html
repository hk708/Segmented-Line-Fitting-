Algorithm Overview

The algorithm takes the input from the user and gives the user the output in the form of a set of equations representing the set of best fit lines for that set of points. The input points are put into a text file for plotting in the visualization. after taking in the set of points it sorts the set of points based on the x value in ascending order. we then parse through the set of points, we store the minimum error for each possible interval of points as well as the slope and intercept of the line of best fit for that interval. we use these values to solve for all the points satisfying i<=N using dynamic programming and storing them in a memoization array(we use the value stored at the index i to calculate the value to be stored in index i+1). we also find the partition points which are the points at which a new line of best fit will be created and these are stored in the segments array. Once we find the optimum segments, we calculate optimum error for the given penalty and the final segment equations with start and ending points using the stored slopes and intercept values, and the error for each segment. An output text file will contain the exact starting and ending points for each of the segments by projecting the points in the input onto the line segment for the purpose of ease of visualization.
